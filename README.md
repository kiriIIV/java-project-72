# Анализатор страниц

**Анализатор страниц** — это веб-приложение для комплексного анализа сайтов на соответствие базовым требованиям поисковой оптимизации (SEO). Оно помогает веб-мастерам и маркетологам быстро выявлять критические ошибки на страницах, которые могут негативно влиять на ранжирование в поисковых системах.

Проект представляет собой полноценный веб-сайт, построенный на легковесном фреймворке Javalin.

## Как пользоваться

1.  Перейдите по ссылке на [работающее приложение](https://java-project-72-vitp.onrender.com) или откройте локальную версию.
2.  Введите URL сайта (например, `https://example.com`), который вы хотите проанализировать.
3.  Нажмите кнопку **"Проверить"**.
4.  Дождитесь завершения анализа.
5.  Изучите детальный отчет с рекомендациями по улучшению SEO-оптимизации вашей страницы.
